# Half-Chess

A two-player Half-Chess application developed in C# using Windows Forms.

## Features

- Custom Half-Chess game implementation
- Interactive graphical board
- Move validation and game logic
- Player management
- Game state handling

## Technologies

- C#
- Windows Forms
- Object-Oriented Programming

## Key Concepts

- Object-Oriented Programming (OOP)
- Game Logic Implementation
- State Management
- Move Validation



## Project Type

Academic software development project.
